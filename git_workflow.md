# Github Basis

## Pull

Pour récupérer les données en ligne.

```bash
git pull
```

## Add

Pour ajouter un fichier que tu veux commiter.

```bash
git add <nom_du_fichier>
git add . # Pour ajouter TOUS les fichiers modifiés
```

## Commit

Pour enregistrer les fichiers ajoutés avec `add` dans un commit.

```bash
git commit -m "Message décrivant les modifications"
```

## Push

Pour envoyer tes changements en ligne.

```bash
git push
```

## Status 

Pour voir les fichiers modifiés, ajoutés, ou supprimés.

```bash
git status
```
